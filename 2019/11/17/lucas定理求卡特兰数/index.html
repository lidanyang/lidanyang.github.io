¶Lucas定理用来求大组合数

  定理表达式为:

  Lucas定理可以按照下面这种方式来理解Lucas定理:A、B是非负整数，p是质数。AB写成p进制：A=a[n]a[n-1]...a[0]，B=b[n]b[n-1]...b[0]。则组合数C(A,B)与C(a[n],b[n])*C(a[n-1],b[n-1])*...*C(a[0],b[0]) modp同余。

  求组合数C(a,b)%p时可以利用C(a,b)%p =a! / ( b! * (a-b)! ) mod p=( a! / (a-b)!) * ( b! )^(p-2) mod p来求，这个公式可以用费马小定理推出。

  最后可以用Lucas(n,m,p)=c(n%p,m%p)*Lucas(n/p,m/p,p)递归求解C(n,m)%p。
