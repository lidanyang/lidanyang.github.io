    求两个单词的最短”距离”

    主要思想:

    word1向word2转化时，如果从前往后调整word1以靠近word2的话，那么word1的最后一个字符的处理只有如下三种情况：

    1. word1的末尾添加一个和word2末尾相同的字符，则问题转化为求word1和word2[:-1]的最短距离；
    2. word1的末尾替换成和word2末尾相同的字符，则问题转化为求word1[:-1]和word2[:-1]的最短距离；
    3. word1的末尾删除一个字符，则问题转化为求word1[:-1]和word2的最短距离；

    3. 为什么找最短要按这个步骤？或者说为什么以循环对word1最后一个字符加删改的方式搜索下去，如果遇到word2那么搜索的深度就是最短距离？严格的数学证明并没有看到，可能只是凭感觉或者说是直觉，觉得照这个步骤来搜索就是最短距离，但是还是感觉心里不踏实，接受程度60%，待以后看到更清晰的解释；
