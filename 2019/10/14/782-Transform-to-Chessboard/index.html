782. Transform to Chessboard

Difficulty: Hard

An N x N board contains only 0s and 1s. In each move, you can swap any 2 rows with each other, or any 2 columns with each other.

What is the minimum number of moves to transform the board into a “chessboard” - a board where no 0s and no 1s are 4-directionally adjacent? If the task is impossible, return -1.

Note:

  • board will have the same number of rows and columns, a number in the range [2, 30].
  • board[i][j] will be only 0s or 1s.

解法

  1. 先用board[0][0] ^ board[i][0] ^ board[0][j] ^ board[i][j]判断数据合法性，即是否任意两行互为取反或者互等，另一种说法是不同的行向量有两个，不同的列向量的个数也只有两个。其实只要不同的行向量有两个的话，那么不同的列向量一定也只有两个，因为翻转任意一行，不会改变两个列向量之间的关系。
  2. 计算其中一个行向量和列向量的个数，要满足不同的行向量的个数能大致相等；
  3. 计算不在正确位置上的0,1的个数，这个数一定要是个偶数；
  4. 根据n的奇偶性，分情况计算总共的不在其位的0,1的个数，将这个数除以2得出交换次数最小值，因为每一次有效的交换（,奇1和偶0或奇0和偶1）会使这个数减2，所以要除以2；
