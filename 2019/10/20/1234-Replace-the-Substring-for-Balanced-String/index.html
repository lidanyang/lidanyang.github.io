1234. Replace the Substring for Balanced String

Difficulty: Medium

You are given a string containing only 4 kinds of characters 'Q', 'W', 'E' and 'R'.

A string is said to be balancedif each of its characters appears n/4 times where n is the length of the string.

Return the minimum length of the substring that can be replaced with any other string of the same length to make the original string s balanced.

Return 0 if the string is already balanced.

解法

  隐藏的双指针题，找这样的[i,j]区间，该区间范围之外的Q,W,E,R的计数小于等于s.length / 4，并且i++或者j–会使Q,W,E,R其中之一的计数超过s.length / 4。

¶总结

  假设题目中给定一个要满足的条件R，则双指针问题的特征是：1.是一个关于求满足某个条件R的区间[l,r]的问题；2.如果一个区间不满足条件R，则该区间的任何子区间均不满足条件R。3.如果一个区间满足条件R，则该区间的任何父区间均不满足条件R。该题中条件R就是：最大的"make string balanced"区间，区间满足条件R，则父区间违反了最大；区间不满足条件R，则子区间违反"make string balanced"。
