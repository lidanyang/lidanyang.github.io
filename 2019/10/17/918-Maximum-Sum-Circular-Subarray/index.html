918. Maximum Sum Circular Subarray

Difficulty: Medium

Given a circular array C of integers represented by A, find the maximum possible sum of a non-empty subarray of C.

Here, a circular array means the end of the array connects to the beginning of the array.  (Formally, C[i] = A[i] when 0 <= i < A.length, and C[i+A.length] = C[i] when i >= 0.)

Also, a subarray may only include each element of the fixed buffer A at most once.  (Formally, for a subarray C[i], C[i+1], ..., C[j], there does not exist i <= k1, k2 <= j with k1 % A.length = k2 % A.length.)

解法

¶己解

¶Next Array

  将问题看成求连续子数组或者求由以A[0]开头和A[A.length() - 1]结尾的两个子数组组合成的子数组中和最大的一个；

¶Prefix Sums + Monoqueue

  将求A中的循环子数组看做求A+A中的连续子数组（子数组长度不超过A），如果利用前缀和之差求连续子数组和的话，问题就可以转化成求以某位置为右端点、长度固定的区间内的最小值，而单调栈也使用来维持一个区间的所有的最值候选值（按照我的理解，实质上单调栈维护的是一个线的包络，而最值肯定会落在包络上）

¶动态规划法

  将求跨边界的子数组的最大值看做是求整个数组和减去连续子数组的最小值（或者加上原数组取负后的连续子数组的最大值）,并且要注意减去或加上的连续子数组不能是整个原数组，要不然会使得到的跨边界子数组为空数组。
