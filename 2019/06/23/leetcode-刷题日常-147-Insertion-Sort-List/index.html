147. Insertion Sort List

Difficulty: Medium

Sort a linked list using insertion sort.


A graphical example of insertion sort. The partial sorted list (black) initially contains only the first element in the list.
With each iteration one element (red) is removed from the input data and inserted in-place into the sorted list

Algorithm of Insertion Sort:

  1. Insertion sort iterates, consuming one input element each repetition, and growing a sorted output list.
  2. At each iteration, insertion sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there.
  3. It repeats until no input elements remain.

¶法二

  观察法一的排序过程，可以发现，就算head的值刚好大于有序链表的最后一个值也要从头结点重新遍历一遍整个链表再插入尾部，这样会浪费大量的时间，效率很低，实际上法一的执行时间是2000ms左右也证明了这一点。所以，可以记录头结点和尾结点cur，每次插入完成后，保证尾结点cur的next结点就是下一个要插入的结点就能省去大量时间。如果下一个结点值如果刚好比尾结点cur的值大，那么就可以直接更新尾结点cur为下一个结点；否则的话，就按照普通的插入排序从头开始找最后一个比要插入的结点的值小的结点，插在该结点之后，并且将要插入结点之后的链表重新连在尾结点cur之后，保证cur.next可以取到下一个待插入的结点。

¶法三

  在法二的基础上，还有可以优化的余地。保存上一次搜索的插入位置p，前一个插入位置p之后的点的值大于下一个要插入的点的值，那么说明新的插入位置在前一个插入位置p之前，那么p重置为头结点（dummy），从头开始搜索；否则，新的插入位置肯定在p之后，那么就只需从p之后开始搜索，这样就可以缩小要搜索的结点数。代码只需在法二上稍作修改：
