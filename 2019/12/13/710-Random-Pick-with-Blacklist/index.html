¶710. Random Pick with Blacklist

Difficulty: Hard

Given a blacklist B containing unique integers from [0, N), write a function to return a uniform random integer from [0, N) which is NOT in B.

Optimize it such that it minimizes the call to system’s Math.random().

Note:

  1. 1 <= N <= 1000000000
  2. 0 <= B.length < min(100000, N)
  3. [0, N) does NOT include N. See .

¶Solution

  在[0,N-1-bl.size()]和[0,N-1]的黑名单之外的所有元素做一个重映射，bl[0]~bl[1]-2映射到bl[0]+1~bl[1]-1，间距为1，起点为bl[0]；bl[1]-1~bl[2]-3映射到bl[1]+1~bl[2]-1，间距为2，起点为bl[1]-1；bl[2]-2~bl[3]-4映射到bl[2]+1~bl[3]-1，间距为3，起点为bl[2]-2，…以此类推可知bl[i]-i~bl[i+1]-i-2映射到bl[i]+1~bl[i+1]-1。
