¶1156. Swap For Longest Repeated Character Substring

Difficulty: Medium

Given a string text, we are allowed to swap two of the characters in the string. Find the length of the longest substring with repeated characters.

¶Solution

  check是当前字符(i所指的字符)的前一个字符和当前正在检查的字符(c所指的)是否匹配的标志位，c是当前正在检查的字符，j是其下标位置。

  1. 当check为false且s[i]!=c，则说明在前一个失配位置加一个c最长只能到i，此时从j开始找到下一个要检查的字符；
  2. 当check为true且s[i]==c则，说明和j到i是一段连续的相同字符的区间，更新res；
  3. 当check为true且s[i]！=c，说明前一段相同字符的连续区域在这里断掉，此时设置check为false，意味着接下来要检查下一段相同字符的连续区域，此时也更新res（因为单段的连续区域也符合要求）。

  res = min(i-j+1, cnt[c-'a']);的更新很巧妙，考虑了是否有多余的c填充空位的情况。
