Difficulty: Medium

Given two integers A and B, return any string S such that:

  • S has length A + B and contains exactly A 'a' letters, and exactly B 'b' letters;
  • The substring 'aaa' does not occur in S;
  • The substring 'bbb' does not occur in S.

Example 2:

Note:

  1. 0 <= A <= 100
  2. 0 <= B <= 100
  3. It is guaranteed such an S exists for the given A and B.

¶Solution

¶法一

  不妨设A>B，则可以每次添一组"aab"，这样每次A减2，B减1，直到A和B相等时就可以每次添一组"ab"，否则只添加剩下的"a"即可。

¶1286. Iterator for Combination

Difficulty: Medium

Design an Iterator class, which has:

  • A constructor that takes a string characters of sorted distinct lowercase English letters and a number combinationLength as arguments.
  • A function next() that returns the next combination of length combinationLength in lexicographical order.
  • A function hasNext() that returns True if and only if there exists a next combination.

Example:

Constraints:

  • 1 <= combinationLength <= characters.length <= 15
  • There will be at most 10^4 function calls per test.
  • It’s guaranteed that all calls of the function next are valid.

¶Solution

¶法一

  位操作模拟组合的过程，从高位到低位对应字母序由小到大，每一位代表某个字符是否被选入，从1<<characters.size()开始，每次减一，判断是否是一个大小为combinationLength的组合，如果是则为字母序的下一个大的组合。

¶法二

  将从后往前第一个可以变大的字符变成下一个更大的字符，然后将之后的位置排成尽可能小的排列，即每个位置的后一个都刚好比它大的递增序列。
