  在由前一天求后一天时，有很多重复计算。比如假设dp、dp1分别存放前一天和后一天的结果，递推式为dp1[i] = min(dp[j-1]+max(jobs[j]~jobs[i]))。在求dp1[i]的时候，如果jobs[i]之前第一个比jobs[i]大的值为jobs[k]，这时递推式中的max(jobs[j]~jobs[i])这一项就取决于jobs[k]，并且之后的最大值肯定不会再取jobs[i]，相当于递推式变成dp1[i] = min(min(dp[j1-1]+max(jobs[j1]~jobs[k])),min(dp[j2-1]+max(jobs[j2]~jobs[i])))=min(dp[k],min(dp[j2-1]+max(jobs[j2]~jobs[i])))，其中j1,j2满足j1 <= k < j2 <= i，其中用到了之前求过的dp[k]，jobs[k]是之前的第一个大于jobs[i]的值，这里出现了单调栈的使用场景即求数组中第一个大于某数的数。又因为jobs[k]是第一个大于jobs[i]的值，所以jobs[j2]肯定都小于jobs[i]，所以递推式中的min(dp[j2-1]+max(jobs[j2]~jobs[i]))可以进一步化简成min(dp[j2-1])+jobs[i]，综上dp1[i] = min( dp[k] , min(dp[j2-1]) + jobs[i])，i>=j2>k。

Then, the players take turns starting with the first player.  In each turn, that player chooses a node of their color (red if player 1, blue if player 2) and colors an uncolored neighbor of the chosen node (either the left child, right child, or parent of the chosen node.)

If (and only if) a player cannot choose such a node in this way, they must pass their turn.  If both players pass their turn, the game ends, and the winner is the player that colored more nodes.
