887. Super Egg Drop

Difficulty: Hard

You are given K eggs, and you have access to a building with N floors from 1 to N.

Each egg is identical in function, and if an egg breaks, you cannot drop it again.

You know that there exists a floor F with 0 <= F <= N such that any egg dropped at a floor higher than F will break, and any egg dropped at or below floor F will not break.

Your goal is to know with certainty what the value of F is.

¶法一

  假设dp[i][j]代表i个鸡蛋，j层楼对应的解，则动态规划的递推式为:dp[i][j] = min(max(dp[i-1][k-1],dp[i][j-k]))+1, 1<=k<=j。观察递推式并结合dp[i][j]是关于j的递增序列，可知使dp[i][j]最小的k位于一个V字形的谷底，如下图。

¶法二

  同样是观察递推式:dp[i][j] = min(max(dp[i-1][k-1],dp[i][j-k]))+1, 1<=k<=j，可以发现使得dp[i][j]取最小的k值会随着j的增加而增加(dp[i-1][k-1]不会随着j增加而增加，dp[i][j-k]随着j增加而增加)，所以当计算当前i的所有dp[i][j]时，每次搜索并不需要使k从0开始，只需要从上次的最小开始即可。

¶法三

  数学法:设f(T, K)是扔K个鸡蛋T次能解出的最高楼层。则f(T,K)有递推式:f(T,K)=1+f(T−1,K−1)+f(T−1,K)。

  递推式的解释:假如从某一层开始扔第一个鸡蛋，如果鸡蛋碎了，那么往下最多能确定f(T-1,K-1)层楼；如果鸡蛋没碎，则往上最多能确定f(T-1,K)层楼，总共最多能确定1+f(T−1,K−1)+f(T−1,K)层楼，故得证。

  设g(t,k)=f(t,k)−f(t,k−1)，则由f(T,K)=1+f(T−1,K−1)+f(T−1,K)和f(T,K−1)=1+f(T−1,K−2)+f(T−1,K−1)可知g(t,k)=g(t−1,k)+g(t−1,k−1)，该式可以联想到一个组合恒等式，即 $g(t, k) = \binom{t}{k}$，所以$f(t,k)=\sum \limits_{1 \leq x \leq k} g(t,k)=\sum \binom{t}{k}$。这个公式也有一个组合解释，把找楼层的过程看做一个鸡蛋碎与不碎的长为k的序列，序列中可能有0,1,2,…,k会摔碎，分别对应$\binom{t}{0},\binom{t}{1},\binom{t}{2},…,\binom{t}{k}$种可能性，每一种可能性对应一个楼层，则一共最多能确定$\sum \binom{t}{k}$高的楼（因为其中还包含了鸡蛋不管怎么都摔不碎的情况，所以要减去1），妙！

  所以问题转化成找一个$f(t,k)=\sum \limits_{1 \leq x \leq k} g(t,k)=\sum \binom{t}{k}$刚好大于N的t，因为f(t,k)关于t递增，所以可以用二分的方法求解。注意，算组合数的时候可以用公式$\binom{n}{k} * \frac{n-k}{k+1} = \binom{t}{k+1}$来简化运算过程。

¶我的解法

  上面的优化我一个都没想到，数学方法只能想到f(T,K)=1+f(T−1,K−1)+f(T−1,K)这个递推式，动态规划也只是能发现递推式，所以还是缺少对于问题的洞察力。

458. Poor Pigs

Difficulty: Hard

There are 1000 buckets, one and only one of them is poisonous, while the rest are filled with water. They all look identical. If a pig drinks the poison it will die within 15 minutes. What is the minimum amount of pigs you need to figure out which bucket is poisonous within one hour?

Answer this question, and write an algorithm for the general case.

General case:

If there are n buckets and a pig drinking poison will die within m minutes, how many pigs (x) you need to figure out the poisonous bucket within p minutes? There is exactly one bucket with poison.

Note:

  1. A pig can be allowed to drink simultaneously on as many buckets as one would like, and the feeding takes no time.
  2. After a pig has instantly finished drinking buckets, there has to be a cool down time of _m _minutes. During this time, only observation is allowed and no feedings at all.
  3. Any given bucket can be sampled an infinite number of times (by an unlimited number of pigs).

解法

  一共可以进行t=p/m次测试，每只猪的t次测试可以有t+1种生死状态，而t+1进制数的每一个数位也恰好有t+1种取值。如果有x只猪，每只猪负责测试t+1进制桶编号中的某一个数位的值，那么t次测试就可以唯一的确认(t+1)^x个桶的毒性，使这个值大于等于n即可。

总结

  之前遇到用一次测试确定次品的类似的问题，是用二进制来解决的，这次是推广到t进制就没有想到，所以学习还是要会举一反三才更有效率。
