918. Maximum Sum Circular Subarray

Difficulty: Medium

Given a circular array C of integers represented by A, find the maximum possible sum of a non-empty subarray of C.

Here, a circular array means the end of the array connects to the beginning of the array.  (Formally, C[i] = A[i] when 0 <= i < A.length, and C[i+A.length] = C[i] when i >= 0.)

Also, a subarray may only include each element of the fixed buffer A at most once.  (Formally, for a subarray C[i], C[i+1], ..., C[j], there does not exist i <= k1, k2 <= j with k1 % A.length = k2 % A.length.)

782. Transform to Chessboard

Difficulty: Hard

An N x N board contains only 0s and 1s. In each move, you can swap any 2 rows with each other, or any 2 columns with each other.

What is the minimum number of moves to transform the board into a “chessboard” - a board where no 0s and no 1s are 4-directionally adjacent? If the task is impossible, return -1.

Note:

  • board will have the same number of rows and columns, a number in the range [2, 30].
  • board[i][j] will be only 0s or 1s.

解法

  1. 先用board[0][0] ^ board[i][0] ^ board[0][j] ^ board[i][j]判断数据合法性，即是否任意两行互为取反或者互等，另一种说法是不同的行向量有两个，不同的列向量的个数也只有两个。其实只要不同的行向量有两个的话，那么不同的列向量一定也只有两个，因为翻转任意一行，不会改变两个列向量之间的关系。
  2. 计算其中一个行向量和列向量的个数，要满足不同的行向量的个数能大致相等；
  3. 计算不在正确位置上的0,1的个数，这个数一定要是个偶数；
  4. 根据n的奇偶性，分情况计算总共的不在其位的0,1的个数，将这个数除以2得出交换次数最小值，因为每一次有效的交换（,奇1和偶0或奇0和偶1）会使这个数减2，所以要除以2；

1157. Online Majority Element In Subarray

Difficulty: Hard

Implementing the class MajorityChecker, which has the following API:

  • MajorityChecker(int[] arr) constructs an instance of MajorityChecker with the given array arr;
  • int query(int left, int right, int threshold) has arguments such that:
    • 0 <= left <= right < arr.length representing a subarray of arr;
    • 2 * threshold > right - left + 1, ie. the threshold is always a strict majority of the length of the subarray

Each query(...) returns the element in arr[left], arr[left+1], ..., arr[right] that occurs at least threshold times, or -1 if no such element exists.

Example:

Constraints:

  • 1 <= arr.length <= 20000
  • 1 <= arr[i] <= 20000
  • For each query, 0 <= left <= right < len(arr)
  • For each query, 2 * threshold > right - left + 1
  • The number of queries is at most 10000

题解

¶线段树

  在查询函数中需要注意一个易错点：int find(int root, int left, int right)返回的是left到right之间的主元素，所以find(root<<1, left, right);和find(root<<1|1, left, right);直接返回即可，而find(root<<1, left, ans[root<<1].r);和find(root<<1|1, ans[root<<1|1].l, right);的返回值要在[left，right]这个区间检查，而不是[ans[root].l, ans[root].r]。
