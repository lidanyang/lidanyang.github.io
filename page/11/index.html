¶Lucas定理用来求大组合数

  定理表达式为:

  Lucas定理可以按照下面这种方式来理解Lucas定理:A、B是非负整数，p是质数。AB写成p进制：A=a[n]a[n-1]...a[0]，B=b[n]b[n-1]...b[0]。则组合数C(A,B)与C(a[n],b[n])*C(a[n-1],b[n-1])*...*C(a[0],b[0]) modp同余。

  求组合数C(a,b)%p时可以利用C(a,b)%p =a! / ( b! * (a-b)! ) mod p=( a! / (a-b)!) * ( b! )^(p-2) mod p来求，这个公式可以用费马小定理推出。

  最后可以用Lucas(n,m,p)=c(n%p,m%p)*Lucas(n/p,m/p,p)递归求解C(n,m)%p。

状态压缩dp

发表于 2019-11-15 | 更新于: 2019-11-15 | 分类于 算法 ， 动态规划

¶基本思路

  用二进制位表示状态，通常是二维数组形式:dp[node][state]，node代表当前状态的结点(比如铺地板砖时的第几行，传递货物时传到哪一个人了)，state代表node的不同状态(相应行的铺砖与否的位图，人手中有无货物的位图)，前面说的状态的二进制表示就用state来表示。问题的解通常是用过程结束时对应的dp值表示(铺地板从第一行铺到最后一行，则最后一行的下一行没有地板装覆盖的方案数即是解，传递货物就对应着全部人都拿过货了)。

  状态压缩dp的难点在于找到对应的node以及node的state表示，状态转移的原则一般是状态间是否兼容或者是否能够由一个状态变化到另一个状态。
